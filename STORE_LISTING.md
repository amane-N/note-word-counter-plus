## 詳細説明（Chrome Web Store 掲載用）

```
note Word Counter+ は、note.com の執筆画面・記事閲覧画面に小さなフローティングウィジェットを追加し、文字数・読了時間・見出し数などをリアルタイムで確認できる無料の執筆支援 Chrome 拡張機能です。

【主な機能】

■ 執筆中のリアルタイムカウント
- 文字数（スペース・改行を除いた実質文字数）
- 読了時間（400字/分を基準に自動計算）
- 見出し数（H1〜H6）
- 段落数
- ハッシュタグ数
- 画像数

■ 目標文字数と進捗バー
ポップアップから目標文字数を設定すると、達成率をプログレスバーで表示します。目標達成時はバーが緑色に変わり、一目で分かります。

■ カスタマイズ可能な表示設定
- ウィジェットの表示位置（右下・左下・右上・左上）
- フォントサイズ（小・中・大）
- 表示する項目を個別にオン／オフ

■ 自由に動かせるウィジェット
ドラッグ操作でウィジェットを好きな位置に移動できます。折りたたみ状態にして最小化することも可能です。

■ 閲覧モード対応
note.com の記事ページでも動作し、他の人の記事の文字数・読了時間をさっと確認できます。

【プライバシーとセキュリティ】
- 外部サーバーへの通信は一切ありません
- 収集するユーザーデータはゼロです
- 設定はブラウザのローカルストレージにのみ保存されます
- 要求する権限は "storage"（設定保存用）と "activeTab" のみです
- note.com と editor.note.com 以外のページでは動作しません

【対応ページ】
- https://note.com/ （記事閲覧ページ）
- https://editor.note.com/ （執筆ページ）

完全無料でご利用いただけます。
```

## スクリーンショット手順

Chrome Web Store では以下のスクリーンショットが推奨されます。

### 推奨サイズ
- **1280 x 800 px**（必須）または 640 x 400 px
- 形式: PNG または JPEG
- 最大 5 枚

### 撮影手順

1. Chrome で `chrome://extensions/` を開き、「デベロッパーモード」を ON にして拡張機能を読み込む
2. ウィンドウサイズを 1280x800 に設定する
   - DevTools を開いて「Toggle device toolbar」から指定するか、ウィンドウを手動でリサイズする
3. 以下のページでスクリーンショットを撮影する

### 推奨スクリーンショット構成

| No. | ページ | ポイント |
|-----|--------|---------|
| 1 | editor.note.com（執筆中） | ウィジェットを展開した状態・文字数・読了時間が見える |
| 2 | editor.note.com（執筆中） | 目標文字数プログレスバーが見える状態 |
| 3 | note.com（記事閲覧） | 閲覧モードでウィジェット表示 |
| 4 | ポップアップ画面 | 設定 UI 全体 |
| 5 | editor.note.com | ウィジェットを左上に移動した状態（カスタマイズ性をアピール） |

### 撮影ツール例
- Chrome DevTools の「Capture screenshot」機能
- macOS: Command + Shift + 4
- Windows: Win + Shift + S（Snipping Tool）

## プライバシーポリシー本文

### note Word Counter+ プライバシーポリシー

最終更新日: 2026-05-05

#### 1. 収集する情報

note Word Counter+（以下「本拡張機能」）は、**いかなる個人情報・ユーザーデータも収集しません**。

本拡張機能が保存するデータは以下のみです。

| データ | 保存先 | 内容 |
|--------|--------|------|
| 表示設定 | ブラウザのローカルストレージ（chrome.storage.local） | ウィジェット位置・フォントサイズ・表示項目・目標文字数 |

上記の設定データはお使いのブラウザ内にのみ存在し、外部サーバーには一切送信されません。

#### 2. 外部通信

本拡張機能は、**いかなる外部サーバーへのネットワーク通信も行いません**。fetch、XMLHttpRequest、WebSocket などの通信手段を一切使用しておらず、完全クライアント完結で動作します。

#### 3. 第三者への情報提供

収集するデータが存在しないため、第三者への情報提供は行いません。

#### 4. 要求する権限

| 権限 | 用途 |
|------|------|
| `storage` | 設定（位置・フォントサイズ・表示項目・目標文字数）をブラウザローカルに保存するため |
| `activeTab` | 現在表示中の note.com タブでウィジェットを動作させるため |

本拡張機能は `https://note.com/*` および `https://editor.note.com/*` のページにのみ動作します。他のウェブサイトへのアクセス権限は持ちません。

#### 5. お問い合わせ

プライバシーポリシーに関するご質問は、Chrome Web Store の拡張機能ページに掲載しているサポートページまたはメールアドレスまでお問い合わせください。
